## v1.0.0 - 2026-03-22
### ✨ Added
* Add 'config init' subcommand to generate the initial configuration file
* Add live streaming of agent feedback (like docker-compose) to stderr
* New '--repo' argument to explicitly pass the target repository path natively or via config
* Support overriding built-in non-interactive agent CLI options via 'extra_args' configuration
* Support customizing the code review prompt instructions via '--skill' flag
* Add 'config init' subcommand to generate the initial configuration file
* Inject 'You are agent (name)' and log agent prompts in debug mode
### 🔄 Changed
* Orchestrator decoupled from concrete PR retrieval constraints (no longer requires 'gh' or 'claude' CLI)
* Update default model configurations to gpt-5.2 and claude-4-sonnet
### 🐛 Fixed
* Fix prompt generation to explicitly request PR review upfront and append skill instructions cleanly
* Remove invalid '--no-interactive' flag from default oz arguments
